Objetivos de la Empresa

Objetivo General

Brinda servicios de insumos médicos de uso casero y con receta que se puedan consultar por medio de la web, además de proveer servicios de consultas médicas a la población agendando estas mismas por el sitio web.

Objetivo Especifico

Crear una página web que tenga la capacidad de presentar un catálogo con las dos categorías de uso de insumos médicos (de uso casero y con receta médica) para la población, haciéndolo de manera práctica para los mismos además de brindarles consultas médicas agendables por este mismo sitio web para facilitar y controlar el flujo de consultas dentro de los locales y entregar un servicio profesional.

Datos Generales

QuixFarma es una empresa dedicada a la distribución de insumos médicos de uso casero y también certificados para poderlos utilizar con recetas médicas, desde los años 2000 nos hemos dedicado a ser útiles para la población salvadoreña abasteciendo de medicamento de primera calidad a cada uno de ellos a precios justos. Ahora con su sitio web, se amplían sus servicios para poder entregar una mejor disposición para la población donde quiera que nuestros usuarios y clientes frecuentes se encuentren.

Descripcion de Servicios

QuixFarma ofrece como servicios los catálogos para medicinas de uso casero como la acetaminophen, loratadina, ibuprofeno, aspirina entre otros para poder tratar dolores y alergias comunes, también medicinas certificadas y recetadas por los médicos para tratamientos más delicados como la acitromicina, la ciprofloxacina, clonazepam, sulforineas y demás para tratar diabetes, depresión, ansiedad, desparasitaciones entre otros. También ofrece servicios de consultas médicas para poder checar el estado de salud si ocurre algún percance como un golpe por algún accidente, alguna gripe tratable a nivel local, desparasitación, etc.

Valores

Accesibilidad

  • Precios justos y productos asequibles para la mayor cantidad de personas.
  • Disponibilidad de medicamentos esenciales sin restricciones innecesarias.
  • Plataforma fácil de usar para el agendamiento de citas.

Compromiso con la Salud Pública

  • Ofrecer productos de calidad y seguros, cumpliendo normativas sanitarias.
  • Brindar acceso a consultas médicas para prevenir y tratar enfermedades.
  • Promover el bienestar integral de la población.
